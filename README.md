# CRM-система Cyberbits

## Описание
CRM-система для управления проектами в сфере кибербезопасности. Система позволяет управлять проектами, документами и пользователями, отслеживать прогресс и анализировать данные.

## Технический стек
- Frontend: React.js, TypeScript, Tailwind CSS
- Backend: Node.js, Express.js
- База данных: PostgreSQL
- Контейнеризация: Docker (опционально)

## Установка

### Вариант 1: Установка с Docker

1. Клонируйте репозиторий:
```bash
git clone https://github.com/your-username/cyberbits-crm.git
cd cyberbits-crm
```

2. Создайте файлы .env:
```bash
# server/.env
PORT=5000
DB_HOST=localhost
DB_PORT=5432
DB_NAME=cyberbits_crm
DB_USER=postgres
DB_PASSWORD=your_password
JWT_SECRET=your_jwt_secret

# client/.env
REACT_APP_API_URL=http://localhost:5000
```

3. Запустите с помощью Docker Compose:
```bash
docker-compose up --build
```

### Вариант 2: Установка без Docker

#### Предварительные требования
- Node.js 18.x
- PostgreSQL 15.x
- npm или yarn

#### Шаги установки

1. Клонируйте репозиторий:
```bash
git clone https://github.com/your-username/cyberbits-crm.git
cd cyberbits-crm
```

2. Настройка базы данных:
```bash
# Создайте базу данных
psql -U postgres
CREATE DATABASE cyberbits_crm;
\q
```

3. Настройка сервера:
```bash
cd server

# Установка зависимостей
npm install

# Создайте файл .env
echo "PORT=5000
DB_HOST=localhost
DB_PORT=5432
DB_NAME=cyberbits_crm
DB_USER=postgres
DB_PASSWORD=your_password
JWT_SECRET=your_jwt_secret" > .env

# Инициализация базы данных
npm run init-db

# Запуск сервера
npm run dev
```

4. Настройка клиента:
```bash
cd ../client

# Установка зависимостей
npm install

# Создайте файл .env
echo "REACT_APP_API_URL=http://localhost:5000" > .env

# Запуск клиента
npm start
```
